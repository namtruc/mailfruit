NOUVEAU : Changement de processus pour passer commande

  • Partages le lien One Drive à l'ensemble du groupe
  • Une fois sur la page du catalogue, clique sur le bouton 'Modifier le classeur"
  • Remplis les coordonnées responsables et livraisons
  • En se connectant à son One Drive, chaque personne remplit sa colonne "Qté"
  • Une fois la commande prête, préviens ton groupe que la commande est cloturée
  • Envoie un mail à cmd@fruitstock.eu avec le lien One drive de ton fichier 
(On ne télécharge plus le tableau pour l'envoyer !)


IMPORTANT :

  • A partir de cette semaine les fruits et légumes frais pourront être commandé à partir de 5kg (quantité total). A vous de vous organiser dans le groupe pour atteindre ce minimum. Hors produit vendu à la pièce (ananas, melon, pastèque...), et produits conditionnés (amandes, dattes, olives...)
  • N'hésitez pas à échanger sur le groupe Messenger pour partager vos facilités et difficultés
  • Pour les Pastèques Crimson, melons, ananas préciser dans le mail de commande le nombre de pièces (ex : 3x6kg et 2x9kg)
  • Tu trouveras désormais les recommandations de Rufino soit dans la Newsletter, soit sur la page Facebook "Fruitstock communauté"
  • L'indice du gazole est en légère hausse, ce qui peut expliquer quelques variations sur les FDP/kg
  • Pour toutes les commandes passées avant dimanche 20h, la palette partira dès le lundi
